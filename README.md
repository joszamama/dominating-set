## Getting Started

### Prerequisites

To run and test this project locally, you must have Maven, Java JDK +8 and Git installed.

### Installation

1. Clone the repo:
   ```sh
   git clone https://github.com/joszamama/dominating-set.git
   ```
2. Install the Maven dependencies:
   ```sh
   cd dominating-set
   mvn clean compile
   ```
